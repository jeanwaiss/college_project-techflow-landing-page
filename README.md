# Projeto Ágil: Sistema de Gerenciamento de Tarefas

## Objetivo
Desenvolver um sistema de gerenciamento de tarefas baseado em metodologias ágeis, simulando um projeto real com uso do GitHub para gestão e qualidade.

## Metodologia
Será utilizada uma abordagem ágil híbrida (Scrum + Kanban), com quadro de tarefas no GitHub Projects.

## Nota sobre a Estrutura de Publicação

Para habilitar a publicação automática via **GitHub Pages**, os arquivos principais (`index.html`, `style.css`, `script.js`) foram movidos para a raiz do repositório, e a pasta `src/` foi removida. Isso é uma exigência do GitHub Pages, que requer que o arquivo `index.html` esteja no diretório raiz ou na pasta `/docs` para ser detectado como página inicial.
